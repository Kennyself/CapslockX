# CapslockX — 用户手册

> **版本**: v1.0.0  
> **日期**: 2026-06-04

---

## 简介

CapslockX 将键盘上的 **CapsLock 键** 改造为强大的修饰键。按住 CapsLock 配合其他按键，即可在主键区完成光标移动、文本选择等高频操作，无需把手移到方向键区域。

---

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 x64 |
| 权限 | **管理员权限**（键盘钩子需要） |
| .NET Runtime | 不需要（自包含发布） |

---

## 安装与启动

1. 下载 `CapslockX.exe`
2. **右键 → 以管理员身份运行**
3. 右下角系统托盘出现蓝色圆形 "CX" 图标即表示运行成功

### 开机自启

目前需手动配置，两种方式：

**方法一：启动文件夹**
```
Win+R → shell:startup → 放入 CapslockX.exe 快捷方式
```

**方法二：任务计划程序**（支持管理员权限自动启动）
```
创建任务 → 勾选"使用最高权限运行" → 触发器：登录时 → 操作：启动 CapslockX.exe
```

## CapsLock 行为规则

| 操作 | 效果 |
|------|------|
| **短按** CapsLock（< 0.3 秒） | 正常切换大小写 |
| **长按** CapsLock（> 0.3 秒后松开） | 无反应，不切换大小写 |
| **按住 CapsLock + 其他键** | 触发快捷键，不切换大小写 |

> 按住 CapsLock 超过 0.3 秒才进入修饰键模式。修饰键模式下松开 CapsLock 不会切换大小写。

## 快捷键清单

### 光标移动

| 快捷键 | 功能 |
|--------|------|
| `CapsLock + E` | ↑ 上移 |
| `CapsLock + D` | ↓ 下移 |
| `CapsLock + S` | ← 左移 |
| `CapsLock + F` | → 右移 |
| `CapsLock + A` | Home 行首 |
| `CapsLock + G` | End 行尾 |

### 文本选择

| 快捷键 | 功能 |
|--------|------|
| `CapsLock + Shift + E` | 向上选中 |
| `CapsLock + Shift + D` | 向下选中 |
| `CapsLock + Shift + S` | 向左选中 |
| `CapsLock + Shift + F` | 向右选中 |
| `CapsLock + Shift + A` | 选中到行首 |
| `CapsLock + Shift + G` | 选中到行尾 |

> 按住 CapsLock + Shift，然后反复按 E/D/S/F 可连续选中

### 系统托盘

| 操作 | 功能 |
|------|------|
| 右键托盘图标 | 打开菜单（About / Exit） |
| 点击 About | 显示版本信息和快捷键提示 |
| 点击 Exit | 退出程序 |

## 常见问题

**Q: 运行后没反应？**
A: 必须以管理员身份运行。右键 CapslockX.exe → 以管理员身份运行。

**Q: 托盘图标在哪？**
A: Win11 默认收起新图标，点任务栏 `^` 箭头展开。

**Q: 大小写指示灯会闪吗？**
A: 短按时正常切换，指示灯变化和原生一致。长按/组合键时完全不切换，指示灯不变。

**Q: 能在游戏里用吗？**
A: 目前没有自动检测游戏模式，建议玩全屏游戏前托盘右键 Exit 退出。

**Q: 中文输入法下能用吗？**
A: 可以，CapsLock 修饰键独立于输入法工作。

## 卸载

退出程序（托盘右键 Exit），删除 `CapslockX.exe` 即可。程序不写入注册表，不创建额外文件。
